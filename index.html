RANDOM NUMBER GENERATOR

Math.floor makes sure the number returned is always a whole number. Math.random() picks a random number from 0 up to the chosen number. The equation after the chosen number for Math.random() is the equation being solved by with the chosen number.

Math.floor(Math.random() * 10 * 1)

0

Math.floor(Math.random() * 152 + 4)

0

Math.floor(Math.random() * 5580 - 6)

0

Math.floor(Math.random() * 18544 * 3)

0

Math.floor(Math.random() * 100000 / 9)

0
